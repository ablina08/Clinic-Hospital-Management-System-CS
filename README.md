# Clinic Hospital Management System

## Overview
This project is a Clinic Management System built using C# and PostgreSQL.

## Features
- Patient management
- Doctor management
- Appointment scheduling
- Treatment tracking
- Payment system

## Technologies
- C#
- .NET
- PostgreSQL
- Npgsql

## How to Run
1. Create database
2. Run schema.sql
3. Run sample data
4. Run project
